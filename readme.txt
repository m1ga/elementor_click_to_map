== Description ==

<img src="img/preview.png"/>

Elementor control that will load a Google Maps iframe after clicking "accept" - no external data is loaded before clicking it.

<img src="img/preview.gif"/>


== Requirements ==

* [Elementor](https://trk.elementor.com/26724)

== Installation ==

* install the plugin
* set the google maps link (Google Maps - share - get the iframe src from the embedded link)
* set an info text, button text and color/width settings
